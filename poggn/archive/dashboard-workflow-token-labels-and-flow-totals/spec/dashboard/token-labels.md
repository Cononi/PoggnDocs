# S1. Token Labels

## 규칙

- dashboard workflow Overview token chip label은 `LLM token`, `local token`을 사용한다.
- dashboard workflow Timeline token chip label도 같은 문자열을 사용한다.
- Korean/English locale 모두 동일한 metric label을 사용해 source 의미를 흔들지 않는다.

## 수용 기준

- `LLM 실사용`, `Local 사용`, `LLM actual`, `Local used` 문구는 workflow token chip에서 더 이상 사용되지 않는다.
